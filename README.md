# sub-matcher

这是一个用于自动匹配视频与字幕文件，并将字幕文件重命名为对应视频文件名的 Python 脚本。

## 功能特点

* **智能匹配**：通过正则表达式自动提取文件名中的“集数”进行精准匹配，而不是依赖文件排序。
* **抗干扰**：自动忽略文件名中常见的分辨率（如 1080p、4k）和编码格式（如 x264、10bit）等干扰字符。
* **多格式支持**：
  * 默认支持的视频格式：`.mkv`, `.mp4`, `.avi`, `.rmvb`, `.wmv`
  * 默认支持的字幕格式：`.ass`, `.srt`, `.ssa`, `.vtt`
* **无损重命名**：字幕文件在重命名时会完全对齐视频的主文件名，同时保留其自身的原始扩展名。

## 使用方法

1. 确保你的电脑上已安装 Python 环境。
2. 将 `rename_subs.py` 脚本文件下载或移动到包含视频和字幕的同一个文件夹中。
3. 在该文件夹下打开终端或命令行工具。
4. 运行以下命令：

   ```bash
   python rename_subs.py
